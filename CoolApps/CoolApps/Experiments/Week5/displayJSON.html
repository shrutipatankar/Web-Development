My Json Parsing Experiment

About the experiment

This experiment is to study parsing of json format text using jQuery. I initially placed an external data.json file in the same directory. However, the jQuery $getJSON function did not resolve the file and hence I used an approach to include a variable with json data. I have displayed this data in a tabular format. Equal efforts were put in to know the CSS rquired to make a table look beautiful! Also, I have provided a search box where when a firstname is put and search button is hit, jQuery displays all the data in a list format in a div just below the table. While doing this I used find function of jQuery. This was good for my learning curve to learn parsing of json.
