Bind and Unbind

About the experiment

This experiment is to study jQuery binding and unbinding events.Calling .off() with no arguments removes all handlers attached to the elements. One can pass arguments of events to this function to disable respective event. Also, .on can be used to bind the element with add event handlers. I have achieved this by providing a button. An image appears below the button. When the image is bound with the events, it fades out and fades in again. However, once the binding is set off, no matter if you hover or click on the button, nothing happens. This can be useful in many applications.

Code snippet
$("body").on("click", "#myimage", clickeffect)
$("body").off("click", "#myimage", clickeffect)
