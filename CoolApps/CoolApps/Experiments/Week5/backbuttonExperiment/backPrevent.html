jQuery Effects Experiment

About the experiment

This experiment is to implementation to prevent the user to navigate back to previous page. I found this of a common use in many web applications. In this experiment, I have prevented the user from navigating back by preventing the back button of the browser. Also, initially I thought I could simply disable the browser button but could not do so. Therefore, using jquery I have prevented the user from navigating back instead.

Code snippet
function disableBack() { window.history.forward() }
