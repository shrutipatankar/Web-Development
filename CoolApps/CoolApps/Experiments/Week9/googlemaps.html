Google Maps API Experiment 1

About the experiment

This experiment is to study the google javascript Maps API. I explored this API from google docs. One Such map will reside on my home page to let the user know exactly where he is currently. I will be providing him an option to choose the type of maps he wants to see.
I have created 6 different maps here:

  • Exploring the Lat Lon options
  • Exploring different zoom levels
  • Exploring the types of Maps

Code snippet
function initialize1() {
var map;
var mapOptions = {
zoom: 8,
center: new google.maps.LatLng(-34.397, 150.644)
};
map = new google.maps.Map(document.getElementById('map-canvas-1'), mapOptions);
}
