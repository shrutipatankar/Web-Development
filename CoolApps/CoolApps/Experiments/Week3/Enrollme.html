My Dance Class Enrollment Form with dynamic input fields!

About the experiment

This experiment is to study the javascript language uses. This experiment uses javascript langauge to dynamically add rows in a table based on user input. For Example, the user wants to enroll in three different types of dancing lessons and chooses to enter his own suitable days on which he prefers, then he can choose to enter these details by clicking "Add Dance" button provided. In this way, user can enter any number of rows that he wants. I saw this as a common requirement which may be useful in many cases and wanted to learn to do this.


Code snippet
var table = document.getElementById("danceTable");
var rowCount = table.rows.length;
var row = table.insertRow(rowCount);

var cellDay = row.insertCell(1);
var element2 = document.createElement("input");
element2.type = "text";
element2.name = "textbox[]";
cellDay.appendChild(element2);
