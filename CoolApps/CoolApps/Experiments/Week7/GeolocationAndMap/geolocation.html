Weather Forecast for the day

About the experiment

This experiment is to extend the geolocation functionality to add the map of the current location along with the weather forecast as before. This data now is being fetched from two different APIs. The map that appears grom google maps api and weather details from a weather api.

Code snippet
var geolocation = navigator.geolocation;
var location = geolocation.getCurrentPosition(showLocation);
var img_url = "http://maps.googleapis.com/maps/api/staticmap?center="+latitude+","+longitude+"&zoom=13&size=500x250&sensor=false";
$("#mapdiv").html("<img src='" + img_url + "'>");
