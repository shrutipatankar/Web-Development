My ajax Experiment to pull data from IMDB server

About the experiment

This is my first attempt to try an ajax call. I have created a UI to put in a name of a movie, I pull the data from IMBD server. I have populated the fields like year of the movie and its details. A reset button has also been provided. The movie that matches the entered string is fetched from the server in JSON format. This data is then added through jQuery to the fields respectively.

Code snippet
$.getJSON(url, function (r) { response = r; console.log(response); $("#title").val(response.Title); $("#year").val(response.Year); $("#plot").val(response.Plot); });
