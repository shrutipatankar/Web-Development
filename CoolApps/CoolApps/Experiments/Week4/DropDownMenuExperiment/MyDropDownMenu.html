My Drop down menu Experiment using jQuery

About the experiment

This experiment is to study the power of jquery to handle hovering effects. I have created a drop down menu on hovering over an list. As the sublist has been set to hidden, it initially does not show up. On hovering over the list header, the sublist shows up. It hides on exiting the hover event. This is achieved by setting the visibility to hidden initially for the sub list. The drop down menu shows up by setting this property to visible dynamically.

Code snippet
.typeOfDance{
visibility: hidden;
}
.typeOfDance li{
border-color: white;
border-style:solid;
border-width: 1px;
}
.Dances{
Visibility: visible;
margin: 2px 2px 2px 5px;
text-decoration: none;
}
