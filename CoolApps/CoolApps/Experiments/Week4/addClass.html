Click on seasons buttons to get respective CSS for the div

About the experiment

This experiment is to study dynamic setting of a class to a div. I have created a div with some text in it. The color of the text and the background image is set dynamically on the button click. This caused some problem as the button click did act weirdly. By removing the current class and then adding the new class helped.

Code snippet
$("#season").removeClass("winter_season autumn_season monsoon_season spring_season").addClass("summer_season");
