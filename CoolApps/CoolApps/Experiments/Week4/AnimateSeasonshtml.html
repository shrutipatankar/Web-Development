Animate Seasons

About the experiment

This experiment is to study the effects of jquery animate function. I have used seasons theme to animate the images of the seasons to come up one by one. This has been achieved with chaining of callback functions. The second image animates only when the first stops animating. This way every image animates and heads back to its original position. Click on Animate Seasons button to see this effect

Code snippet
$("#animate").click(function () {
$("#summer").animate({
top: '250px',
height: '+=150px',
width: '+=150px',
}).animate({
top: '0px',
height: '-=150px',
width: '-=150px',
}
