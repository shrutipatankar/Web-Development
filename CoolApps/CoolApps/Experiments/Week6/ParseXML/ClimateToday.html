About the experiment

This experiment is to study parsing of XML which I have done using a weather API. This data had been pulled from the API by appending the name of the city accepted from the user in the parameters. Once I recieve XML, I have pulled values from XML for some of the data that I put. Next, I arranged that on the HTML page along with a good UI.
