HTML5 Content Editable Experiment

About the experiment

This experiment is to view the html5 feature of making editable elements on the html page. I have created an editable notepad here. I intend to make it more of a notepad element by making a bullet point appear everytime an enter button is pressed. However, currently it just appears as a content editable notepad.


Code snippet
<div class="notepad" contenteditable="true">
