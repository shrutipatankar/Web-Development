Gradient Experiment

About the experiment

This experiment is to view the different types of gradients and also eventually view the effect of gradients in terms of a sparkling circle.


Code snippet
.sparkling-circle{
background-image: -webkit-radial-gradient(40px 40px, circle cover,#66ccff, #000000);
-webkit-animation-name:spin;
-webkit-animation-duration: 3s;
-webkit-animation-iteration-count: infinite;
-webkit-animation-timing-function: linear;
}
