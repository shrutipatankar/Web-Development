View different ways of viewing an unordered list

About the experiment

This experiment is to view different ways in which an unordered list can be rendered. I intend to use this set of experiments for my project. This experiment initially renders unordered list normally, than retrieves the unordered list with a symbol, and then proceeds to retrieve different items of an unordered list with different symbols and eventually renders an unordered list as a vertical tab. My home page of the project intends to use such a symbolic representation of a tabbed interface using unordered list.

Code snippet
ul.symbolic li{
padding: 5px 5px 5px 5px;
margin: 2px 2px 2px 2px;
padding-left: 25px;
background-image: url("../../../../myimages/order.png");
background-size: 20px;
background-position: 0px center;
background-repeat: no-repeat;
}
