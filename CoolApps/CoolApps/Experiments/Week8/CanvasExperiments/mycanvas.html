Paintings by Canvas in HTML5

About the experiment

This experiment is to understand the working of canvas in HTML5. After studying SVG, while exploring further, I tried to implement canvas as well. Although, like SVG, canvas does not provide inline tags, I understood the implementation of canvas and also that canvas draws its graphics on the fly using javascript.

Code snippet
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.fillStyle = "pink";
ctx.fillRect(20, 20, 80, 140);
