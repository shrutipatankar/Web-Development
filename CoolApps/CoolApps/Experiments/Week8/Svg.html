Paintings by using inline Svg in HTML5

About the experiment

This experiment is to understand the working of svg in html5. This experiment has been done to implement and understand different shapes using svg. This experiment has been done for sole reason of knowledge and I will not be using this in my project. I was just curious of its working. I have t tried to overlay images, study different shapes, strokes, polygon drawings and gradients in this experiment.

Code snippet
<svg width="100" height="100">
<circle cx="50" cy="50" r="40" stroke="green" stroke-width="2" fill="yellow" />
<rect x="25" y="25" rx="0" ry="0" width="50" height="50" style="fill:green;stroke:brown;stroke-width:3" />
<circle cx="50" cy="50" r="15" stroke="green" stroke-width="2" fill="pink" />
</svg>
