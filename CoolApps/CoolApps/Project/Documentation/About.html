ABOUT PROJECT: This project has been created to help the travelers learn more about the place. I have used following APIs for this project:
  1. Places and Restaurants: Google Places API
  2. Maps: Google Maps API
  3. Weather: Open Weather API
Project Summary

The project home page displays what the project si about. It provides some idea to what to expect out of the project.
The search place page detects the location of the user. It is a single page app that shows restuarant data near by the user by default. Details such as name of the place, address, rating and if the place is open currently is shown to the user. Also, the address of the place appears as a hyperlink, which when clicked takes user to a map locating the place in a new tab. The user may filter the food results based on restaurant, bar, cafe and bakery.
Same features apply when the user is searching for places worth visting.
The climate tab shows the users the details of climate in his area. This is determined by latitude and longitude of the user's current location from Open Weather API. Note that the temperature is shown only in Faranheits.
Apart from these features, the user can see pictures of the place he searching in the frame beside the map. The project has been made CSS responsive to allow cell phone users to see the data clearly. The frame related to the pictures disappears completely though once the window is resized at a certain level as the map then occupies the width of the screen completely.
