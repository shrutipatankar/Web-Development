Climate Tab
Feature Screenshots


The Climate tab shows following data:
  • Temperature in Fahrenheit
  • City (May locate county)
  • Country
  • Weather Details in short
  • Humidity
  • Windspeed
Note that the images that appear next to the data of climate are actually unordered lists. I have enhanced the unordered lists to have different images to show the data. All this data is pulled from OpenWeather API. Note that open weather API provides temperatures only in Kelvin. I have internally converted the Kelvin temperature to Fahrenheit by using following formula:
1.8 * (Kelvin-273)) +32 (found on following site:Kelvin-to-Fahrenheit)
