jQuery Effects Experiment

About the experiment

This experiment is to study the effects that can be achieved by using jQuery. Some of the effects that I have used for the images to vanish are

  • toggle
  • slide up
  • slide down
  • hide
  • fade out
  • fade in
I also learned chaining of these effects. If someone clicks on last two images for examples, they can see that the image will slide up , then slide up and then again slide up. This has been achieved by using the concept of chaining the effects one after another.

Code snippet
$(".hiphop").slideUp('slow');
$(".salsa").slideUp('slow').slideDown('slow').slideUp('slow');
