My iFrame and tabs implementation

About the experiment

This experiment is to demonstrate implementation of iFrames and tabbed interface using li tag. In this experiment, I have created 4-5 forms, each gets displayed in the iframe on selection. Default being home page. This experiment also provides division of the form into two sections next to each other. On left hand side I have provided a form to use html5 input tags. On right hand side resides the images for the snakes and the points on how to identify them etc.


Code snippet

<li> <a onclick="iframeSetSource('Home.html');"> Home </a> </li>
